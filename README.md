# 🚀 JavaScript Practice Journey

Welcome to my JavaScript practice repository! This project documents my learning path, starting from core fundamentals to advanced logic and full-scale web integration.

## 📌 Project Overview
This repository is divided into three main phases:
1.  **Core Basics:** Variables, Data Types, and Operators (Node.js & Console).
2.  **Logic Building:** Loops, Functions, and Array Methods.
3.  **Web Integration:** Connecting JS logic with HTML/CSS (DOM Manipulation).

---

## 🛠️ Tech Stack & Environment
* **Language:** JavaScript (ES6+)
* **Runtime:** Node.js (for backend logic)
* **Browser:** Google Chrome Console (for debugging)
* **Editor:** VS Code

---

## 📂 Repository Structure

### 1. Fundamentals (`/basics`)
Includes foundational programs to understand how JS works under the hood.
* `variables.js`: Difference between `let`, `const`, and `var`.
* `data_types.js`: Strings, Numbers, Objects, and Arrays.
* `operators.js`: Arithmetic and logical operations.

### 2. Logic & Control Flow (`/logic`)
Small programs focusing on problem-solving.
* `conditionals.js`: If-else and Switch case practice.
* `loops.js`: Practice with `for`, `while`, and `forEach`.
* `functions.js`: Standard and Arrow functions.
* `array_methods.js`: Mastering `map()`, `filter()`, `reduce()`, and `splice()`.

### 3. HTML Integration (Coming Soon 🚧)
Practical projects where JS interacts with the browser UI.
* DOM Selection and Events.
* Form Validations.
* Mini Projects (Counter, Todo List, etc.).
